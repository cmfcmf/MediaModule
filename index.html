Upload all sorts of media

The module supports many media types natively (images, plaintext, MarkDown, videos, audio, pdf, …) and further types can be added with ease! Media can be dragged- and dropped and is uploaded in the background.

Embed and add media from the web

Not only can you upload media, you can also search for and embed all kinds of media from the web! Here are some: Tweets, YouTube videos, Music, Flickr images, …

Full watermark support

You can create text- and image-based watermarks to watermark all uploaded images. The watermarks scale automatically and can be exchanged at any time. The original images are preserved.

Automatic module upgrades!

For the first time ever, a module can upgrade itself with just a single click. No need to manually download new versions from GitHub anymore! Receive bug fixes right when they are ready.

Full Scribite integration

Include and upload media right from Scribite's CKEditor. This means you can embed images, tweets, videos, … right from Scribite.

Full Hook integration

Hook single media objects or entire collections to any Zikula content supporting hooks.

ZIP downloads

If you enable it, all media in a collection can be downloaded as ZIP file. Additionally, every medium can be downloaded at it's own.

Automatic metadata extraction!

Regardless whether you upload an image, a video or a pdf file: Metadata is extracted on the fly, filtered and displayed! This includes, but is not limited to: video length and compression, music genre and artist, image size and flash settings.

Beautiful and SEO URLs!

All URLs use the new Zikula "Routing", resulting in URLs like /collections/pictures/cats/f/my-cat as well as /collections/media/new.

Sortable media and collections

All media objects can be sorted inside their collections. Additionally, all collections can be sorted. It is as easy as dragging and dropping them! As easy as a piece of cake!

No legacy, only the latest bits!

The module uses as little of Zikula's legacy code as possible. All templates are using Bootstrap and Twig. The module is service-based and uses Symfony forms. This will make it easier to keep up with the core development.

Legal tools

The module provides you with any features you need these days: Select licenses for every medium, add attribution and authors.

Licenses

Create and edit licenses. You can also hook a license to any other Zikula module using hooks!

HTML, MarkDown or Plaintext

Media and collection descriptions can either be plaintext, markdown or HTML (using Scribite).

Live preview of PDF files

If your browser supports it, PDF files can be viewed directly on-page.

Automatic syntax highlighting

Uploaded plaintext files will be syntax highlighted automatically. Bonus: You can also view them in fullscreen mode.

Rendered MarkDown files

Uploaded MarkDown files will be rendered automatically for you.

Multiple collection templates

You want your images to be displayed in a slideshow? Or a simple table? Or a grid? Now worries, three different templates are included.
